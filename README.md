# 🏠 Landing Page Inmobiliaria - Casa en Venta (Maipú, Mendoza)

Sitio web profesional de alta conversión diseñado para la venta directa de una propiedad residencial. 
Desarrollado bajo metodología ágil enfocado en la velocidad de carga y adaptabilidad total.

🚀 **Ver proyecto en vivo:** https://github.io

## 🛠️ Tecnologías Utilizadas
* **HTML5 y CSS3:** Estructuración limpia y maquetación personalizada.
* **Bootstrap v5.3:** Framework utilizado para garantizar la adaptabilidad en PC, Notebooks, Tablets y Celulares.
* **JavaScript Nativo (Vanilla JS):** Motor lógico personalizado para controlar el carrusel de 15 fotos de manera autónoma sin dependencias externas.

## 🛡️ Características y Seguridad
* **Diseño para Baja Visión:** Fuentes escaladas y optimizadas para facilitar la lectura a personas mayores o cortas de vista.
* **Formulario Seguro:** Integración con pasarela encriptada anti-hackers mediante Formspree, protegiendo los datos del cliente y el correo del propietario contra Phishing y Spam.
* **Optimización de Imágenes:** Control de visualización mediante propiedades avanzadas de CSS (`object-fit: contain`) para evitar recortes accidentales de planos técnicos o fotos verticales.
